
Problem Statement
Find the smallest number such that the sum of it's digits is N and it is divisible by 10N.If no such number exists, print -1
Input section
The first line contains, T, denoting the number of test cases.
Each test case consist of a single integer N.
Output section
For each test case, print the required answer.
Input constraints
1 ≤ T ≤ 100
0 ≤ N ≤ 10000

Subtasks

Subtask #1 (5 points) : 0 ≤ N ≤ 6
Subtask #2 (25 points) : 0 ≤ N ≤ 18
Subtask #3 (70 points) : Original Constraints

Sample Input
1
4

Sample Output
40000
