
You are given a sequence A1,A2,…,ANA1,A2,…,AN. You have to find the length of a positive-sum subarray such that the sum of elements of the subarray is minimum. If there is more than one answer then print the largest possible answer.
NOTE: If no subarray satisfies the given property than print "-1".
Input:

The first line of the input contains a single integer TT denoting the number of test cases. The description of TT test cases follows.
The first line of each test case contains a single integer NN.
The second line contains NN space-separated integers A1,A2,…,ANA1,A2,…,AN.

Output:
For each test case, print a single line containing one integer — the length of the positive-sum subarray.
Constraints

1≤T≤51≤T≤5
1≤N≤1051≤N≤105
0≤Ai≤1090≤Ai≤109 for each valid ii

Sample Input:
2
4
1 0 1 2
3
1 2 3

Sample Output:
2
1
