
There is a candy shop that sells NN varieties of candies. The ii-thth variety has aiai candies.
You want to buy as many candies as possible. If you buy bibi candies of variety ii, then for all 1≤j<i1≤j<i either bj=0bj=0 (you bought 0 candies of type jj) or bj<bibj<bi (you bought less candies of type jj than of type ii).
Remember, there is no shortage of money.

Input

The first line contains an integer NN, denoting the number of varieties of candies.
The next line contains NN integers aiai, denoting the number of candies of each variety.
Output
Print the maximum number of candies you can buy.

Constraints

1≤N≤2⋅1051≤N≤2⋅105
1≤ai≤1091≤ai≤109
Sample Input:
5
1 3 2 5 8
Sample Output:
16
Explanation
It is optimal to buy 0+1+2+5+80+1+2+5+8 candies.
